This is a [Next.js](https://nextjs.org/) project.

## Getting Started

First, install packages:

```bash
npm install
# or
yarn
```

Second, build the project:

```bash
npm run build
# or
yarn build
```

Third, run the server:

```bash
npm run start
# or
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
